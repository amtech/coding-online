# coding-online 
  在线生成系统原型，适合一般的管理系统（eg. sms,cms），帮助完成95%的基础代码。
  
  系统基础架构： 1. 开源框架 springmvc + hibernate ；
                 2. 系统缓存 redis; 
                 3. 前面页面 easyui + bootstrap + jquery + lhgdialog .
  
  

  
  系统还有部分没有开发完成，还需要一段时间。
